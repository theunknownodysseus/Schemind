## 🛠️ Getting Started

### ⚙️ Prerequisites

- Node.js (v14 or later)
- npm or yarn

### 📦 Installation

```bash
git clone https://github.com/theunknownodysseus/Schemind.git
cd schemind
npm install
```

### ▶️ Running in Development

```bash
npm run dev
```

Visit: [http://localhost:5172](http://localhost:5172)

### 🏗️ Build for Production

```bash
npm run build
npm start
```
